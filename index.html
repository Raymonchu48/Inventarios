Guía rápida

  1. Crea un proyecto en Supabase.
  2. Ejecuta schema.sql y luego seed.sql.
  3. Pega la URL y la anon key arriba.
  4. Sube esta carpeta a GitHub y activa GitHub Pages.
  5. Abre tu web y empieza a trabajar.

Para endurecer seguridad, el siguiente paso es añadir autenticación y políticas por usuario.
